Introduction

People who are blind or visually impaired (BVI) use screen readers to assist their computer usage. The linear nature of these screen reader interactions makes understanding or navigating hierarchical structures like nested menus or tabular content on the web difficult for BVI users. To improve the experience of non-visually navigating nonlinear data, Spatial Region Interaction Techniques (SPRITEs) was introduced as a novel method for navigating two-dimensional structures using the keyboard. This NVDA add-on implements specifically the table navigation techniques for browsing tables with the Google Chrome browser.
