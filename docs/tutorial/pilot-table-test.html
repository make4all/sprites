Pilot Study Table

You are given a weekly schedule for a college-level calculus class, and your task is to use SPRITEs table navigation to find out some important information from the schedule.

  1. You want to figure out which weeks have exams so you could prepare for it ahead of time. How would you locate the weeks with exams?
  2. There are some weeks in the quarter where there are no classes on Mondays. How would you find those weeks?
